# Food Delivery Data Analysis

## Project
Analysis of food delivery data to understand customer behavior, restaurant performance, and revenue patterns.

## Files
- `analysis.ipynb` - Main notebook with all code
- `final_food_delivery_dataset.csv` - Combined dataset

## Analysis
1. Order trends
2. Customer behavior  
3. City performance
4. Cuisine analysis
5. Membership impact
6. Revenue patterns

## Tools
- Python, Pandas, NumPy
- Matplotlib, Seaborn
- Jupyter Notebook

## Key Findings
- Best quarter: Q3
- Top city: Bangalore  
- Popular cuisine: Mexican
